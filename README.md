# JavaUI Studio

JavaUI Studio is a modern, lightweight, and open-source IDE specifically designed for developing Java graphical user interfaces. 

Unlike traditional IDEs, JavaUI Studio focuses on providing the best possible experience for visual interface creation through native integration between design, code, and artificial intelligence. It combines the simplicity of tools like Figma with the power of a professional Java development environment.

## 🚀 Vision
**"The best specialized IDE for Java UI development."**
The goal is a "Visual-First" approach where the interface is designed visually and clean Java code is generated automatically.

## 🛠️ Core Tech Stack
- **Framework:** Tauri (Desktop)
- **Frontend:** React + TypeScript + Vite
- **Backend:** Rust
- **Styling:** TailwindCSS + shadcn/ui
- **Editor:** Monaco Editor
- **Canvas:** HTML Canvas / Konva.js / React Flow
